  # Idle Clicker

Небольшой браузерный кликер с системой перерождений и древом постоянных улучшений.

## Геймплей

- Клик по кнопке даёт монеты.
- Монеты тратятся на апгрейды клика и авто‑добычу.
- При перерождении прогресс рана сбрасывается, игрок получает кристаллы.
- Кристаллы тратятся в отдельном меню бустов на постоянные улучшения (криты, множители, общий доход).

## Стек

- HTML, CSS, JavaScript (без фреймворков).
- localStorage для сохранения прогресса.

# Idle Clicker

A small browser clicker game with a prestige system and a permanent upgrade tree.

## Gameplay

- Clicking the main button earns coins.
- Coins are spent on click upgrades and auto‑miners.
- On prestige, the current run resets and the player gains crystals.
- Crystals are spent in a separate boosts screen on permanent upgrades (crit chance, crit multiplier, global income).

## Tech stack

- HTML, CSS, vanilla JavaScript.
- localStorage for saving progress.
